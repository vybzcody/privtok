# PrivTok - Private Content Platform

## Overview
**PrivTok** is a privacy-first content sharing and subscription platform built on Aleo blockchain. It enables creators to 
share encrypted content with subscribers while maintaining privacy and control over their digital assets. The platform 
demonstrates how to build a decentralized creator economy using Aleo's zero-knowledge capabilities.

## Features

### 🎨 Creator Features
* Create and manage encrypted content with privacy-first approach
* Set up subscription tiers with flexible pricing models
* Receive tips and payments privately or publicly
* Control content access through smart contract permissions

### 👥 Subscriber Features
* Subscribe to creators with private or public payments
* Access encrypted content through secure decryption
* Send private messages and tips to creators
* Manage subscriptions and view content history

### 🔒 Privacy Features
* End-to-end encryption for all content and messages
* Zero-knowledge proofs for subscription verification
* Private payment processing with optional public visibility
* Secure key management and content access control
